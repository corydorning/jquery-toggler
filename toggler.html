Toggler Plugin Dependencies

To use toggler, it is recommended that you use jQuery version 1.8+. Reference the jQuery library followed the toggler JavaScript file:

<!-- load jQuery library -->
<script src="/common/js/libs/jquery-1.8.min.js"></script>

<!-- load toggler plugin -->
<script src="jquery-toggler.js"></script>

Now you are ready to use toggler! Below is some example code on the ways you can use it.

Toggler Plugin Options

Options via HTML5 Data Attribute

Data-Attribute Value Description
data-toggler-duration
  • 400
 Effects the duration of the animation.

NOTE: any numeric value in milliseconds will work. EX: 1000 = 1 second.
data-toggler-slide
  • True
  • False
 Controls the direction of the animation. True slides the content vertically and false slides the content horizontally.
data-target
  • .class-name
  • #id-name
 Targets the class or id on the content container to make it collapsible.

Options via JavaScript Property

Property Value Description
duration
  • 400
 Effects the duration of the animation.

NOTE: any numeric value in milliseconds will work. EX: 1000 = 1 second.
slide
  • True
  • False
 Controls the direction of the animation. True slides the content vertically and false slides the content horizontally.
target
  • $(.class-name)
  • $(#id-name)
 Targets the class or id on the content container to make it collapsible.

Example

$('.toggler').toggler({ duration: 400, slide: true, target: $('.target-content') });
